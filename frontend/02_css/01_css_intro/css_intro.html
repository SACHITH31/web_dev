INLINE CSS:

Inline CSS is a way of applying CSS styles directly inside a particular HTML element using the style attribute. It affects only that single element.

style="property: value;"

<p style="color: red; font-size: 18px;">Hello World</p>

INTERNAL CSS:

Internal CSS is written inside the <style> tag within the <head> section of the HTML document. It affects only that single HTML page.

<style>
  p { color: blue; }
</style>

<p>This paragraph is styled using internal CSS</p>

EXTERNAL CSS:

External CSS is written in a separate .css file and linked to the HTML document using the <link> tag. It is used to style multiple pages.

<link rel="stylesheet" href="style.css">

/* style.css */
p { color: green; }
