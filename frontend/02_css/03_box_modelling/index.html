Box Modelling In CSS

This page demonstrates the CSS box model, which includes margins, borders, padding, and the content area.

content (WIDTH * HEIGHT)

Width: when ever you set the width of an element, and when we decrease the screen size then it will gets overflowed.

Max-Width: when ever you set the max-width of an element, and when we decrease the screen size then it will gets adjusted according to the screen size.

Height: when ever you set the height of an element, and when we decrease the screen size then it will gets overflowed down wards.

Max-Height: when ever you set the max-height of an element, and when we decrease the screen size then it will gets adjusted according to the screen size.

Overflow: the overflow property specifies what should happen if content overflows an element's box. It can be set to visible, hidden, scroll, or auto.

Notes: In the above example, when the content exceeds the maximum height of 1000px, the overflow property set to "hidden" ensures that the excess content is not visible, effectively hiding it from view.

Overflow: auto - This setting adds scrollbars to the element only when the content exceeds the element's dimensions. If the content fits within the element, no scrollbars are shown.

Overflow: visible - This is the default value. Content that overflows the element's box will be visible outside of it.

Overflow: scroll - This setting always shows scrollbars, regardless of whether the content fits within the element or not.
