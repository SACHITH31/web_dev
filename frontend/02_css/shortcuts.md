# ShortCuts in web-development
- For html files use `!` and press tab to generate boilerplate code.
- To link a css file use `link:css` and press tab.
- To link a js file use `script:src` and press tab.
- div#child$*4  will generate 4 divs with id child1, child2, child3, child4. After clicking tab or enter you will see below code:
```html
<div id="child1"></div>
<div id="child2"></div>
<div id="child3"></div>
<div id="child4"></div>
```
- div.parent>div.child*3 will generate below code:
```html
<div class="parent">
    <div class="child"></div>
    <div class="child"></div>
    <div class="child"></div>
</div>
```
