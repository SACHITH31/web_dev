Welcome To Inline and Block Elements

Inline Elements: Inline Elements are those which do not take up the full width of their container and do not start on a new line.

Examples:

- span element. - a element. - strong element. - b element. - small element. - i element. - u element. - img element.

Block Elements: Block Elements are those which take up the full width of their container and start on a new line.

Examples:

- div element.

- p element.

- heading elements.

    - ul element.
  • - li element.
    • - section element.
    - header element.
    - footer element.

    inline-block in css:

    - The inline-block value in CSS is used to create elements that have characteristics of both inline and block elements. Elements with display: inline-block; can sit next to each other on the same line (like inline elements) but can also have width and height properties applied to them (like block elements).
