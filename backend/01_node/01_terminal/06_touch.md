# touch Command

## What is `touch`?

Used to create new empty files.

---

# Syntax

```bash
touch file_name
```

---

# Example

```bash
touch app.js
```

Creates a new file called `app.js`.

---

# Multiple Files

```bash
touch a.txt b.txt c.txt
```

Creates multiple files.

---

# Notes

- `touch` creates empty files
- Very useful while creating projects
- Fast way to create files from terminal
