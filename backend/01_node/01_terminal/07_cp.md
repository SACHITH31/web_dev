# cp Command

## What is `cp`?

Used to copy files and folders.

---

# Syntax

```bash
cp source destination
```

---

# Example

```bash
cp app.js copy.js
```

Copies `app.js` into `copy.js`.

---

# Diagram

```text
app.js
   │
   └── copied → copy.js
```

---

# Notes

- `cp` = Copy
- Original file stays unchanged
- Useful for backups and duplicates
- Can also copy folders with `-r` flag (recursive)
- Example: `cp -r folder1 folder2` copies `folder1` into `folder2` (creates `folder2` if it doesn't exist)
