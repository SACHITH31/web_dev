# cd Command

## What is `cd`?

Used to change folders in terminal.

---

# Syntax

```bash
cd folder_name
```

---

# Example

```bash
cd Documents
```

Moves into the `Documents` folder.

---

# Example

```bash
cd ..
```

Moves one folder back.

---

```bash

```

---

# Notes

- `cd` = Change Directory
- Used for folder navigation
- One of the most used terminal commands
- `cd ..` moves backward
