# rm Command

## What is `rm`?

Used to delete files and folders.

---

# Syntax

```bash
rm file_name
```

---

# Example

```bash
rm app.js
```

Deletes `app.js`.

---

# Delete Folder

```bash
rm -r folder_name
```

Deletes a folder recursively.

---

# Warning

Deleted files cannot be restored easily.

Be careful while using `rm`.

---

# Notes

- `rm` = Remove
- Permanently deletes files
- Powerful but dangerous command
- Use with caution, especially with `-r` flag for folders
