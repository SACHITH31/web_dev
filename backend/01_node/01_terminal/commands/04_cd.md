# cd Command

## What is `cd`?

Used to change folders in terminal.

---

# Syntax

```bash
cd folder_name
```

---

# Example

```bash
cd Documents
```

Moves into the `Documents` folder.

---

# Example

```bash
cd ..
```

Moves one folder back.

---

# Example

```bash
cd ../..
```

Moves two folders back.

---

# Relative Path Navigation

```bash
SACHITH@DESKTOP-F0N8OVB MINGW64 ~/Desktop/022/web_dev/backend (main)
$ cd ../frontend
```

### Output

```bash
SACHITH@DESKTOP-F0N8OVB MINGW64 ~/Desktop/022/web_dev/frontend (main)
$
```

Moves from `backend` folder to `frontend` folder.

---

# Home Directory

```bash
cd ~
```

or

```bash
cd
```

Moves to the home directory.

---

# Folder Names With Spaces

```bash
cd "My Folder"
```

Used when folder names contain spaces.

---

# Previous Directory

```bash
cd -
```

Moves to the previously visited folder.

---

# Multiple Folder Navigation

You can move through multiple folders in a single command.

---

# Example

```bash
SACHITH@DESKTOP-F0N8OVB MINGW64 ~/Desktop/022/web_dev (main)
$ cd backend/01_node/01_terminal
```

---

# Example

```bash
SACHITH@DESKTOP-F0N8OVB MINGW64 ~/Desktop/022/web_dev (main)
$ cd "backend/01_node/01_terminal"
```

Both work correctly.

---

# Symbols

| Symbol | Meaning |
| ------ | ------- |
| `.` | Current folder |
| `..` | Previous folder |
| `~` | Home folder |
| `/` | Root folder |

---

# Diagram

```text
web_dev
├── backend
└── frontend
```

If you are inside `backend`:

```bash
cd ../frontend
```

You move to `frontend`.

---

# Notes

- `cd` = Change Directory
- Used for folder navigation
- One of the most used terminal commands
- `cd ..` moves backward
- Quotes are useful when folder names contain spaces
- `/` is used to move through folders
- `~` takes you to the home directory
- `-` takes you to the previous directory
- `.` refers to the current directory
- `..` refers to the parent directory.
