# cp Command

## What is `cp`?

Used to copy files and folders.

---

# Syntax

```bash
cp source destination
```

---

# Example

```bash
cp app.js copy.js
```

Copies `app.js` into `copy.js`.

---

# Diagram

```text
app.js
   │
   └── copied → copy.js
```

---

# Copy Into Folder

```bash
cp app.js src/
```

Copies `app.js` into the `src` folder.

---

# Multiple Files

```bash
cp app.js style.css src/
```

Copies multiple files into `src`.

---

# Relative Path Example

```bash
cp app.js ../
```

Copies `app.js` one folder back.

---

# Rename While Copying

```bash
cp app.js server.js
```

Creates a copied file with a new name.

---

# Hidden File Example

```bash
cp .env .env.backup
```

Creates a backup copy of `.env`.

---

# Folder Copy

```bash
cp -r folder1 folder2
```

Copies `folder1` into `folder2`.

---

# Multiple Commands Example

```bash
mkdir backup && cp app.js backup/ && ls backup
```

### What Happens?

1. Creates `backup` folder
2. Copies `app.js` into it
3. Shows files inside `backup`

---

# Notes

- `cp` = Copy
- Original file stays unchanged
- Useful for backups and duplicates
- `-r` is used for copying folders
- Commonly used in backend development
