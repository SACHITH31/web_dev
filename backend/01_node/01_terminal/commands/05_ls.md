# ls Command

## What is `ls`?

Shows files and folders inside the current directory.

---

# Syntax

```bash
ls
```

---

# Example

```bash
ls
```

### Output

```bash
index.html
style.css
app.js
```

---

# Why is it Useful?

- View files quickly
- Check folder contents
- Useful while navigating

---

# Notes

- `ls` = List
- Shows files and folders
- Does not open files
- Very commonly used command
