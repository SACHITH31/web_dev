# pwd Command

## What is `pwd`?

`pwd` stands for:

```bash
Print Working Directory
```

Shows the current working folder path.

---

# Why is `pwd` Important?

When working in terminal, sometimes you move into many folders.

So you may forget:

- Where you currently are
- Which folder you are working in
- Which project folder is opened

The `pwd` command helps you know your current location.

---

# Basic Syntax

```bash
pwd
```

---

# Simple Example

```bash
pwd
```

### Output

```bash
/home/sachith/Desktop/project
```

This means:

You are currently inside the `project` folder.
