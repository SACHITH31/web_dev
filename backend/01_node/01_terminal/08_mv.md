# mv Command

## What is `mv`?

Used to move or rename files and folders.

---

# Syntax

```bash
mv old_name new_name
```

---

# Rename Example

```bash
mv app.js main.js
```

Renames `app.js` to `main.js`.

---

# Move Example

```bash
mv app.js src/
```

Moves `app.js` into the `src` folder.

---

# Notes

- `mv` = Move
- Used for moving and renaming
- Original file changes location/name
- Useful for organizing files and folders
- Can also move folders with the same syntax (e.g., `mv folder1 folder2` moves `folder1` into `folder2`)
