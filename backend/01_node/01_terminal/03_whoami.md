# whoami Command

## What is `whoami`?

Shows the current logged-in user name.

---

# Syntax

```bash
whoami
```

---

# Example

```bash
whoami
```

### Output

```bash
sachith
```

This means the current user is `sachith`.

---

# Why is it Useful?

- Helps identify current user
- Useful while using servers
- Commonly used in Linux systems

---

# Notes

- `whoami` = Who Am I
- Shows only the username
- Safe command
- Useful for beginners while learning terminal
